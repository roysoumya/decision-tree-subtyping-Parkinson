# MDS Dataset
Please download the dataset from [https://github.com/ferjorosa/parkinson-subtypes/tree/main/data](https://github.com/ferjorosa/parkinson-subtypes/tree/main/data). If you want to publish the results from this data, please request permission from the Movement Disorders Society, by emailing at [info@movementdisorders.org](info@movementdisorders.org). More details are available at [https://mds.movementdisorders.org/publications/rating_scales/request_form.php](https://mds.movementdisorders.org/publications/rating_scales/request_form.php).
